The Basic Laguage of the web: HTML

Posted by Laura Jones on Monday, June 21st 2027

All modern websites and web applications are built using three foundamental technologies: HTML, CSS and JavaScript. These are the languages of the web.

In this post, let's focus on HTML. We will learn what HTML is all about, and why you should learn it.

What is HTML

HTML stands for Hypper Text Markup Language that the web developer use to structure and describe the content of a webpage (not a programming language).

HTML consist of elements that describe different types of content: paragraph, links, headings, images, video,etc. Web browser understand HTML and render HTML code as websites.

In HTML, each element is made up of

  1. The opening tag
  2. The closing tag
  3. The actual element

You can learn more at MDN Web Docs.

Why should you learn HTML

There are countless reason for learning the fundamental language of the web. Here are 5 of them:

  • To be able to use the fundamental web dev language.
  • To hand-craft beautiful websites instead of relying on tools like Wordpress or Wix.
  • To build web applications.
  • To have fun.
